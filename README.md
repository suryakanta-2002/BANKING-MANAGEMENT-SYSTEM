# BANKING-MANAGEMENT-SYSTEM
It is a mini project of BANKING MANAGEMENT SYSTEM using python programming language with Exception handling and pickling and unpickling concept of file handling. The project contains the following.
1. Add New Customer (Opening an Account)
2. Delete Customer ( Closing an Account)
3. WithDraw
4. Deposit
5. View Customer Details
6. View All Customer Deatils
7. Search Customer Details
8. Pin Generate
9. Pin Update
10. Exit
